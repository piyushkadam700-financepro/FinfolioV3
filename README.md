# Piyush Kadam — Finance Portfolio V3

Live: [piyushkadam700-financepro.github.io/FinfolioV3](https://piyushkadam700-financepro.github.io/FinfolioV3)

Personal finance portfolio built with vanilla HTML, CSS & JavaScript. Design and animation language inspired by [landonorris.com](https://landonorris.com).

## Animations
- Preloader counter (0→100%) with slide-away transition
- Scroll-pinned giant text — words illuminate one by one on scroll
- Per-line clip-path heading reveals with GSAP ScrollTrigger
- Horizontal scroll banner driven by scroll position
- SVG signature draw-on animation
- Magnetic hero buttons that follow the cursor
- Full-page nav overlay with staggered link reveals
- Elastic cursor ring with lag follower
- Dual marquee tickers that accelerate with scroll velocity
- Parallax SVG contour backgrounds

## Stack
Vanilla HTML · CSS · JavaScript · GSAP 3 + ScrollTrigger · Google Fonts (Bebas Neue, Playfair Display, DM Sans) · No frameworks · No build tools
